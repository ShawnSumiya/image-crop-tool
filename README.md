# 画像余白自動クロップツール

ReactとTailwind CSSを使用した、画像の余白を自動で検出して切り取るツールです。

## 機能

- **複数画像のドラッグ＆ドロップ**: 一度に複数の画像をアップロード可能
- **自動余白検出**: 透明ピクセルや白い余白を自動で検出
- **しきい値調整**: スライダーで余白検出の感度を調整可能
- **非同期処理**: 何十枚もの画像を同時に処理
- **個別ダウンロード**: 処理済み画像を個別にダウンロード
- **ZIP一括ダウンロード**: すべての処理済み画像をZIPファイルで一括ダウンロード

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 使い方

1. ブラウザでアプリケーションを開く
2. しきい値スライダーで余白検出の感度を調整（必要に応じて）
3. 画像をドラッグ＆ドロップするか、クリックしてファイルを選択
4. 処理が完了すると、プレビューリストに表示されます
5. 個別にダウンロードするか、「すべてダウンロード」ボタンでZIPファイルとして一括ダウンロード

## 技術スタック

- React 18
- Vite
- Tailwind CSS
- JSZip
